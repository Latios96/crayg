## Building
### Linux
```shell
mkdir build
cd build
cmake ..
make
```
### Run the tests
```shell
cd build
make test
```

### Windows (experimental)
```shell
mkdir build
cd build
cmake -G "Visual Studio 15 2017 Win64" ..
```
